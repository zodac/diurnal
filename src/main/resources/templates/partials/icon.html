Named icon catalogue — the single home for every inline SVG glyph, so call sites reference an icon by name instead of embedding unreadable path data, and a glyph is reused (or redrawn) in one place. Every icon is decorative (aria-hidden) — the HOST control carries the accessible name (aria-label / visible text). The one deliberate exception to this catalogue is partials/eye-icons.html: its two glyphs form a stateful show/hide pair wired to data-eye-show/data-eye-hide toggling, so they stay in their own component partial.
