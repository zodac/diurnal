{! Shared page footer — build year, version and source link. Self-contained: it pulls everything from the {inject:appInfo} CDI bean (net.zodac.diurnal.web.AppInfo), so it can be dropped onto any page with no data passed in. Full-bleed like the navbar (partials/navbar.html): a `bg-surface` bar with a top border, content padded by px-4. The three items — year, version, GitHub — sit centred on one row, separated by `·` delimiters. It stays a single row on every viewport (no mobile stacking). Every colour is a semantic token, so it themes automatically. Standard in-flow sticky footer: it's the last child of each page's `.page-shell` wrapper (after ). Because the wrapper is at least one viewport tall and
is `flex-1`, the footer's bottom rests on the bottom of the viewport on short pages, and sits just below the content (reached by scrolling) on tall ones. !}
