{! Shared calendar toolbar — the prev/next/year/today buttons, the month/year title, the "jump to month" icon, and the month/year picker popup. Rendered identically for EVERY calendar style (full, minimal, stacked) so the navigation chrome is the same across calendars, themes and viewports. dashboard.html wires the stable button ids to the active calendar: FullCalendar's API for the `full` view, the grid-render functions for `minimal`/`stacked`. The popup is positioned absolutely against the enclosing `#calendar-wrap` (which is `position: relative`). !}
