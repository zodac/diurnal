{! Shared trailing options cell for an *editable* data-table row (see CLAUDE.md "Editable tables"). View state ([data-dt-view]): Edit + Delete. Edit state ([data-dt-edit], hidden until dtStartEdit): Save + Cancel. Built from primitives so any table can reuse it without leaking entity specifics: id — the entity id rowPrefix — DOM-id prefix of the row, so the row is "#{rowPrefix}-{id}" (e.g. "action", "user-row") formPrefix — id prefix of the row's edit
, submitted by Save (e.g. "action-form", "user-form") confirmBase — URL base for the confirm-delete GET, i.e. "{confirmBase}/{id}/confirm-delete" Edit is opened/closed by the shared dtStartEdit/dtCancelEdit toggles defined in layout.html. !}
