{! A single selectable preview tile for the Settings Theme / Calendar-style / Font pickers: a radio-driven selectable frame around a fixed-size screenshot thumbnail (rendered by `partials/preview-thumb.html`, which owns the consistent per-viewport sizing/cropping), with an `(!)` button that opens the FULL (uncropped) screenshot in the lightbox. The lightbox derives its image from whichever thumbnail `` is currently visible (the thumbnail and full image are the same file — the thumbnail is just CSS-cropped), so this tile only needs to carry the title + group for the modal, not explicit src attributes. Exactly one of `themeBase` / `calStyle` / `fontBase` selects the thumbnail mode (the others ""); all three are forwarded to preview-thumb (Qute renders strict, so each must always be present). Params: group - radio group name ("theme" | "calendarView" | "font") value - this option's value current - the user's currently-selected value (drives the checked state) label - short caption shown under the thumbnail title - heading shown in the full-size preview modal alt - image alt text themeBase - "light"|"dark"|"system" → Theme tile; "" otherwise calStyle - "full"|"minimal"|"stacked" → Calendar tile; "" otherwise fontBase - "nova"|"standard" → Font tile; "" otherwise !}
